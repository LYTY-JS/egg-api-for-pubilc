基础信息

API基础URL: http://[YOUR_DOMAIN_OR_IP]/api

所有请求都需要在Header中添加: Content-Type: application/json

1. 处理静态图片

Endpoint: GET /process-image?imageUrl=URL

请求参数

参数 类型 说明
imageUrl string 要处理的图片URL

示例请求

GET http://[YOUR_DOMAIN_OR_IP]/api/process-image?imageUrl=https://example.com/photo.jpg
注意： 图片URL必须是公开可访问的。

2. 处理GIF动画

Endpoint: GET /processGif?gifUrl=URL

请求参数

参数 类型 说明
gifUrl string 要处理的GIF URL

示例请求

GET http://[YOUR_DOMAIN_OR_IP]/api/processGif?gifUrl=https://example.com/animation.gif
注意： 处理GIF可能需要较长时间，取决于GIF的大小和帧数。

3. 客户端代码示例

JavaScript Fetch示例

// 处理静态图片
fetch('http://[YOUR_DOMAIN_OR_IP]/api/process-image?imageUrl=' + encodeURIComponent(imageUrl))
  .then(response => response.json())
  .then(data => console.log(data));

// 处理GIF
fetch('http://[YOUR_DOMAIN_OR_IP]/api/processGif?gifUrl=' + encodeURIComponent(gifUrl))
  .then(response => response.json())
  .then(data => console.log(data));
