GET /process-image?imageUrl=URL

处理静态图片，将原图中的人脸替换为默认替换图片。

请求参数

参数 类型 是否必需 描述
imageUrl string 是 原图的URL地址

请求示例

GET /process-image?imageUrl=https://example.com/original.jpg

响应

200 成功响应
{ "status": "success", "data": { "processedImageUrl": "https://[YOUR_DOMAIN_OR_IP]/output/processed.jpg", "processingTime": 1.24 } }
GET /processGif?gifUrl=URL

处理GIF动画，将每一帧中的人脸替换为默认替换图片。

请求参数

参数 类型 是否必需 描述
gifUrl string 是 原GIF的URL地址

请求示例

GET /processGif?gifUrl=https://example.com/animation.gif

响应

200 成功响应
{ "status": "success", "data": { "processedGifUrl": "https://[YOUR_DOMAIN_OR_IP]/output/processed.gif", "processingTime": 3.57, "frameCount": 24 } }
